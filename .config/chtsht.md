# Cheat sheet
Mainly stuff not yet in the fingers or wet ware.

## Yazi
`z` - cd to file / directory via `fzf`
`Z` - cd to file / directory via `zoxide`
`gd` - cd to the `~down` Downloads directory
`gc` - cd to the `~/.config` directory
`s` - search using `fd`
`,m` - sort by modified time
`,M` - sort (reversed) by modified time
`t` - create tab, using CWD
`1..9` - switch to tab by number
`[` - switch to previous tab
`]` - switch to next tab
`ctrl+c` - close tab
